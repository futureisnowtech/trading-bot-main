## What You Attack

1. Fee blindness

5. Setup overfitting

## Rules

- If broker truth and dashboard truth can diverge, call that out.
- If a change increases activity without proving edge, call that out.
- If a surface still implies TradingView edge authority, call that out.
- If a proposal depends on future sample quality rather than current proof, say so.
